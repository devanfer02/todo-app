# 📝 Todo App

Just trying to learn fullstack with building simple web app 🙂

## Setup
1. run command ```npm install```
2. run command ```chmod +x cmd/serve.sh```
3. run command ```cmd/serve.sh``` to instantly run the api
4. run command ```npm run dev```

## Tech Stack
- Vite + React
- Typescript
- Golang
- Gin
- MySQL
